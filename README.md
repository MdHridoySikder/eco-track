# Project Name

EcoTrack — Sustainable Living Community

## Project Description

EcoTrack is a community platform for eco-conscious individuals to discover and join sustainability challenges, share practical eco-tips, browse local green events, and track personal environmental impact. The platform focuses on measurable, community-driven progress.

Users can explore:

- Challenges: Participate in various environmental initiatives.
- Tips: Share and view practical eco-friendly tips.
- Events: Find and join local green events.
- Personal Progress: Track individual contributions and achievements.

### Features

- Browse and join challenges by category.
- View challenge details including duration, impact metrics, and participants.
- User authentication (Sign in / Sign up / Logout) with Firebase.
- Profile management (update name and photo).
- My Activities dashboard to track personal progress.
- Responsive design for desktop and mobile devices.
- Interactive navigation using React Router.

### Technologies Used

- **React** – Frontend framework
- **React Router** – Page routing
- **Tailwind CSS** – Styling and responsive design
- **Daisy UI** – Prebuilt Tailwind components
- **Firebase** – Authentication and user management
- **MongoDB** – Database for challenges, tips, events, and user progress
- **Express.js** – Backend API server
- **Node.js** – Server runtime
- **Vercel** – Server hosting
- **Netlify / Surge** – Client hosting
- **React Toastify** – Notifications
- **React Icons** – UI icons
- **Fetch API** – API calls
- **Git & GitHub** – Version control
- **VS Code** – Code editor
- **Google Fonts** – Custom web fonts
